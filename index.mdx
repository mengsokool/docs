BeeNut is an open-source computer-vision counting kiosk that transforms a desktop machine, Raspberry Pi, or single-board computer into a reliable counting appliance. Bring your own YOLO ONNX model, configure your target parts, and BeeNut handles the live camera preview, stable counted totals, and hardware controls — all from a touch-friendly interface.

## What BeeNut Does

BeeNut runs two coordinated processes: a native C++ service (`beenutd`) that handles camera capture, YOLO inference, GPIO control, and frame transport, and a Flutter operator UI that displays the live preview, count totals, and settings. Everything is packaged into a single installable bundle.

## Supported Platforms

| Platform | Package | Notes |
|---|---|---|
| Raspberry Pi / Debian ARM64 | `beenut_*_arm64.deb` | Appliance kiosk mode with flutter-pi |
| Linux Desktop AMD64 | `beenut_*_amd64.deb` | Desktop or appliance mode |
| macOS | `BeeNut-macos-*.dmg` | Drag-to-Applications installer |

## Get Started in 3 Steps

<Steps>
  <Step title="Install BeeNut">
    Run the one-command Linux installer or download a package from the [releases page](https://github.com/mengsokool/app.beenut/releases).

    ```bash
    curl -fsSL https://raw.githubusercontent.com/mengsokool/app.beenut/master/scripts/install-linux.sh | sudo bash
    ```
  </Step>
  <Step title="Add Your YOLO Model">
    Place your `yolo.onnx` file (and an optional `labels.txt`) in the models directory, then select it from the BeeNut settings panel.

    ```
    service/models/my-model/
    ├─ yolo.onnx
    └─ labels.txt
    ```
  </Step>
  <Step title="Start Counting">
    Launch BeeNut, select a target from the catalog, point your camera at the parts, and read the live count on screen.
  </Step>
</Steps>
